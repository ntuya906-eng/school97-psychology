Нийслэлийн ерөнхий боловсролын 97 дугаар сургууль
Сурагчийн сэтгэлзүйн бүртгэлийн систем

Энэ хувилбар нь:
- Сурагч бүртгэл хийх
- Өдөр, цаг сонгох
- Бүртгэлийн нууц код авах
- Сурагч өөрийн код + утсаар зөвхөн өөрийн цагийг харах
- Админ бүх бүртгэлийг харах, өөрчлөх

====================================
1. VS Code дээр ажиллуулах
====================================
1) ZIP файлаа задлана.
2) VS Code дээр задалсан хавтсыг Open Folder хийнэ.
3) Terminal нээнэ.
4) Дараах командыг ажиллуулна:

py -m venv venv
venv\Scripts\activate
pip install -r requirements.txt
python app.py

5) Browser дээр:
http://127.0.0.1:5000

====================================
2. Админ нэвтрэх хэсэг
====================================
Админ хуудас:
http://127.0.0.1:5000/admin/login

Анхны админ мэдээлэл:
Нэвтрэх нэр: admin
Нууц үг: School97@2026

Анхаарах зүйл:
Ашиглаж эхлэхээсээ өмнө app.py файл доторх DEFAULT_ADMIN_PASSWORD-ийг солихыг зөвлөе.
Эсвэл environment хувьсагчаар ADMIN_PASSWORD тохируулж болно.

====================================
3. Сурагч өөрийн цагийг харах
====================================
Сурагч бүртгэл амжилттай болсны дараа систем код өгнө.
Жишээ: PSY-8A12BC

Тэр код + өөрийн утасны дугаараар:
http://127.0.0.1:5000/check
хуудас руу орж зөвхөн өөрийн уулзалтын мэдээллээ харна.

====================================
4. Файлууд
====================================
app.py                -> үндсэн Python код
school97.db           -> SQLite өгөгдлийн сан (анхны ажиллуулахад автоматаар үүснэ)
templates/            -> html хуудаснууд
static/               -> css, зураг

====================================
5. Хэрэв алдаа гарвал
====================================
Хэрэв python command ажиллахгүй бол:
py app.py

Хэрэв library алга гэвэл:
pip install -r requirements.txt
